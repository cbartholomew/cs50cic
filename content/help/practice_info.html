What are Practice Sets?

Practice psets infuse cs50 lectures with short Arduino hands-on exercises that can be finished fairly quickly. Practice psets provide students who prefer a more “hands-on” approach, while practicing the syntactical beauty that of C.

All practice sets must be completed within the Arduino IDE using student versions of the sketches that will be provided for each practice pset.

In order to install the Arduino IDE on appliance, one should follow the instructions here.
